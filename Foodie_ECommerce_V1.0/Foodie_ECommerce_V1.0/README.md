        # Foodie E-commerce (V1.0)

Estructura de proyecto creada como scaffold profesional con patrón MVC.

## Instrucciones rápidas

1. Copia `.env` y rellena las credenciales.
2. Crea un entorno virtual y instala dependencias (Flask, google-cloud-firestore, etc.).

```bash
python -m venv .venv
source .venv/bin/activate
pip install Flask
```

3. Ejecuta la app:

```bash
python run.py
```
